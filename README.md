# Proyecto_Nomina_Calculadora
Este proyecto, denominado Nomina Intelligence Hub, consiste en el desarrollo de una herramienta avanzada de auditoría de nómina y cumplimiento tributario para el año 2026. Está diseñada bajo una visión de dirección contable, enfocándose en la precisión técnica y la generación de reportes gerenciales detallados.

Los pilares fundamentales del sistema son:

Cálculo Integral de Nómina: Procesa devengos, horas extras detalladas, recargos y auxilios de transporte proporcionales a los días laborados.

Depuración Tributaria Automatizada: Realiza la depuración de la base laboral aplicando normativas vigentes (Art. 55, 126, 206 y 387 del ET) para calcular la retención en la fuente mediante Procedimiento 1 o 2.

Gestión de Carga Prestacional: Calcula provisiones mensuales, aportes a seguridad social y parafiscales, validando automáticamente la exoneración del Art. 114-1 del ET.

Reporte Gerencial Multiformato: Permite la exportación de resultados en archivos CSV, Excel (optimizado con openpyxl) y PDF (generado con fpdf2), facilitando la trazabilidad de los costos para la empresa y el desprendible para el empleado.

Infraestructura de Desarrollo Profesional: El proyecto se gestiona en un entorno WSL (Linux) sincronizado con GitHub mediante llaves SSH, garantizando un control de versiones robusto y seguro.
